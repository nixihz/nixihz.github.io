    python 数据分析怎么回事

    python 只是一门脚本语言，真正让它拥有各种能力的是各种各样的包。

    而数据分析基本用到的是这个包：

    Pandas

    pandas 是一个快速、强大、灵活且易于使用的开源数据分析和操作工具。

    分析数据，拢共分几步？

    获取原始数据 → 加工数据 → 输出数据。

    如果你有固定的数据源，比如团队里固定格式的 excel 报表，那么 pandas 是非常适合你的。

    使用 pandas 本质上是编写程序，也就是加工数据这一步，虽然 excel 也能做，但是重复的劳动没人愿意做，懒是第一生厂力。既然是程序了，那就能做到重复使用，更改不同的数据源就可以快速得出结果。同时又因为是程序，使得过程非常便于调试，更容易找出出错的步骤。

    这里得出 2 个优点：

    1. 可重复执行，平均成本降低。
    2. 便于调试，加工时间降低。
