    Hugo Expand 展开和隐藏

    Expand 扩展可以通过隐藏部分文本来帮助减少屏幕混乱。通过单击扩展内容。

    示例

    默认

    1
2
3
4

    {{< expand >}}
## Markdown content
Lorem markdownum insigne...
{{< /expand >}}

    自定义标签

    1
2
3
4

    {{< expand "这是自定义标签" "更多" >}}
## Markdown content
Lorem markdownum insigne...
{{< /expand >}}
