    2019-10-01
    tool, vscode

    文字中的空白到底是tab和空格？

    开发测试过程中，我们可能需要构造各种tab分割的文本数据，然而默认配置下，输入tab，就被转化成了4个空格，为了避免这些坑，我们来尝试做些处理。

    显示空白

    设置中搜索renderWhitespace，设置为all

    RenderImage when len(FileIDs) == 0 NYI

    这个时候，就能看出空白到底是tab还是空格，如图：

    RenderImage when len(FileIDs) == 0 NYI

    这一步操作不仅能显示tab和空格，而且区分了 tab 和 空格

    • 拷贝文本，保留tab
    • 输入文本，替换tab为4个空格
